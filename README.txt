거울게임(MirrorGame)

실행파일: index.html

필요환경: 웹 브라우저

조작방법:
[History Panel (좌)]
- QUERY 버튼을 눌러 StartPoint n을 입력
[Submit Board (우)]
- 거울 조각을 드래그하여 보드에 옮길 수 있음
- 드래그 도중 R키를 눌러 반시계 방향으로 90도 회전
- 보드 위의 숫자에 마우스를 올려 제출 배치의 레이져 경로를 확인
- 보드 위의 숫자를 클릭하여 StartPoint n을 설정
- SUBMIT 버튼을 눌러 제출 (QUERY 한번 당 한번의 제출 기회)

디버깅: 브라우저 개발자 콘솔을 열어 정답 확인 가능
